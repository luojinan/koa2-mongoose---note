- 克隆项目后安装依赖包
  - 根目录下
  ```
  cnpm i
  ```
  - server文件夹下
  ```
  cnpm i
  ```
- 启动项目，需要同时在server文件夹下，和根目录下启动
  - 根目录下
  ```
  npm run dev
  ```
  - server文件夹下
  ```
  npm start
  ```
- 访问项目
  - 自动打开
  ```
  http://localhost:9528
  ```
  - 手动打开后端
  ```
  http://localhost:9000
  ```
- 效果 
  - 在vue-element-admin登录页面，打开调控台用个接口，返回了数据
